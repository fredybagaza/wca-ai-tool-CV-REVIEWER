
#WHAT THE PROJECT DOES
A CV Reviewer automatically analyzes a resume (CV) and gives feedback.

##In simple terms:
👉 You upload a CV (PDF/image)
👉 The system reads it
👉 Then it evaluates it and gives suggestions or a score

#PURPOSE OF THE PROJECT THE MAIN GOAL IS TO:

Help people improve their CVs, Match CVs with job descriptions
Save time for recruiters by automating screening

⚙️ How It Works (Step-by-Step)
1. Input CV User uploads a CV (PDF or image) PyPDF2 → read PDF text
CV content vs job description

2.Output / Feedback
The system shows: Matching score, missing skills, Suggestions (e.g. “Add more experience in Python”)
